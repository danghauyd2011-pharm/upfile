// HƯỚNG DẪN SETUP
BƯỚC 1 — Tạo Bot:
1. Mở Telegram → tìm @BotFather
2. Gõ /newbot
3. Đặt tên bot (vd: MyFileBot)
4. Copy token dạng 123456:ABCdef...

BƯỚC 2 — Lấy Chat ID (chọn 1 trong 2):
Cách A — Chat riêng với bot (dễ nhất):
1. Tìm bot vừa tạo trên Telegram
2. Bấm Start
3. Mở link này (thay TOKEN):
https://api.telegram.org/botTOKEN/getUpdates
4. Copy số "id" trong phần "chat"

Cách B — Dùng Group/Channel:
1. Tạo group/channel mới
2. Thêm bot vào làm admin
3. Gửi 1 tin nhắn bất kỳ vào group
4. Mở link getUpdates như trên
5. Chat ID dạng -100xxxxxxxxxx
